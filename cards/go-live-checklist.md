# The Go-Live Card

Clear every gate before the agent posts to anything real.

- [ ] **Least privilege.** The agent's entitlements are the smallest set the job needs, and nothing more.
- [ ] **A real human gate.** Approval is a real person reviewing, not an auto-approve left over from testing.
- [ ] **Audit on and provable.** Every action is logged with the agent's identity, and the hash chain verifies.
- [ ] **The rules match your books.** Allowed accounts, currencies, and tolerances are yours, not the samples.
- [ ] **The data path is agreed.** Security has signed off on where documents go when the model reads them.
- [ ] **A dry run first.** Full run against a sandbox or test client, audit trail read end to end.
- [ ] **A way to watch it.** The KPIs, the override rate, and the control numbers are wired up.
- [ ] **A kill switch.** One control stops the agent path without stopping the business process, and a named person holds it.

For the full pre-flight inspection, work through Appendix D: The Tenant Validation Checklist.
